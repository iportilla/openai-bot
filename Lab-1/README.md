# Lab 1 — Basic OpenAI Chatbot

## Goal

This lab introduces the fundamentals of interacting with the OpenAI API. You will learn how to send messages to a model and receive responses in a conversational loop.

## Files in this folder

- `example.py` — Minimal demonstration of sending a message and printing a response.
- `generic_openai_chatbot.py` — A simple interactive chatbot loop where the user enters input and the model replies.

## How to Run

```bash
cd Lab-1
python example.py
```

or

```bash
streamlit run generic_openai_chatbot.py
```

## Key Concepts Introduced

- How to call the OpenAI Chat API
- User → Model → Response flow
- System / User / Assistant roles in chat messages
